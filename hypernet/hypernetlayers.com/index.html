Through years of research, network analysis, and testimonials from insiders, we have identified at least five distinct layers of the Hypernet, each more restricted and mysterious than the last.

Layer 1: Surface Web
Public websites, search engines, standard protocols
Layer 2: Deep Web
Password-protected sites, private databases, unindexed content
Layer 3: Dark Web
Encrypted networks, anonymous browsing, hidden services

Government networks, military communications, corporate backchannels

Detailed Layer Analysis

LAYER 1: SURFACE WEB (Public Internet)

Access Level: Public
Estimated Size: 2.5 billion pages
Access Method: Standard browsers (Internet Explorer, Netscape)

This is the internet most people know. Search engines like Yahoo!, AltaVista, and the new Google service index these pages. Contains commercial websites, news sites, personal homepages, and public forums.

Key Characteristics:

  • Indexed by search engines
  • Accessible via standard HTTP/HTTPS
  • No special software required
  • Subject to government monitoring

LAYER 2: DEEP WEB (Hidden Internet)

Access Level: Restricted
Estimated Size: 500x larger than Surface Web
Access Method: Direct URLs, credentials, special queries

The Deep Web contains password-protected sites, private databases, internal company pages, and academic repositories. Much larger than the surface web but mostly benign.

Content includes:

  • Corporate intranets
  • Academic databases
  • Private email servers
  • Banking systems
  • Medical records

LAYER 3: DARK WEB (Anonymous Networks)

Access Level: Anonymous Tools Required
Estimated Size: Unknown
Access Method: TOR, Freenet, specialized browsers

WARNING: Dark Web access may trigger government monitoring systems

The Dark Web requires specialized software for access and focuses on anonymity. Originally developed by the US Navy for intelligence operations, now used by various groups for both legitimate privacy and illegal activities.

Notable characteristics:

  • Onion routing for anonymity
  • Encrypted communications
  • Hidden service addresses (.onion domains)
  • Heavily monitored by law enforcement

LAYER 4: CHARTER WEB (Institutional Networks)

Access Level: Classified
Estimated Size: Classified
Access Method: Government/Military clearance required

This layer was first documented by former NSA contractor Michael Harrison in 1999 before his disappearance. Charter Web represents closed networks used by governments, military organizations, and major corporations.

Alleged contents:

  • ARPANET legacy systems
  • Military communication networks
  • International intelligence sharing
  • Corporate espionage networks
  • Scientific research collaboration

Access Level: QUANTUM ENCRYPTION
Estimated Size: UNKNOWN
