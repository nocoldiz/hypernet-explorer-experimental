JSON Description Line Break Converter

Instructions

This tool adds line breaks (\n) in the middle of skill descriptions in your RPG Maker MV/MZ skills JSON file.

  1. Adjust the minimum character count setting if needed
  2. Click "Choose File" to select your skills JSON file (e.g., Skills.json)
  3. The tool will automatically process the file
  4. You can preview sample modifications below
  5. Click "Download Modified JSON" to save the updated file
